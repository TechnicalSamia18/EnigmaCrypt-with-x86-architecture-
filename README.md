# EnigmaCrypt-with-x86-architecture-
EnigmaSuite v1.4 is a console-based cryptographic application written entirely in x86 assembly language
using the MASM assembler and the Irvine32 library. The system simulates the core operating principles of
the historic Enigma cipher machine — the electromechanical encryption device used extensively during
World War II — and transposes them into a modern software implementation that operates on plaintext
files. The project demonstrates the practical application of low-level programming concepts including
register management, memory addressing, procedure design, and Windows API interaction
A python based GUI was added for better assesibility and usage.
